# pkoscik dotfiles

A _mostly_ sane set of configuration files. 

## Installation

The dotfiles are managed by __GNU Stow__:

```
cd ~
git clone https://github.com/pkoscik/dotfiles
cd dotfiles
stow <directory name>
```
